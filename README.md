# Getting Started 

### 1. Clone the repo
```
git clone https://github.com/AtharvaSDeshpande/Smart-Classroom.git
```

### 2. Change directory
```
cd Smart-Classroom
```

### 3. Install dependencies
```
npm install
```

### 4. Run the app
```
npm start
```
